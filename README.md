 ## Project Overview

This project presents an HR Analytics Dashboard built with Power BI to analyze employee data at Palmoria Group. The dashboard provides insights 
into employee demographics, salary distribution, departmental structure, and performance ratings.  
The aims of the project is to help HR managers and decision-makers understand workforce trends, identify salary patterns, and employee performance to support better organizational planning.

## Dashboard Preview 

The dashboard visualizes key HR metrics, including:

- Total employees

- Total salary paid

- Average salary

- Gender distribution

- Employee count by department

- Salary comparison by gender and department

- Performance rating distribution

 - Salary distribution by location

## Business Value  

This dashboard helps HR and management to:
- Monitor gender balance in the organization
- Identify salary differences between employees
- Understand the number of employees in each department
- Track changes in employee performance
- Support HR in making data-based decisions

 ## Tools Used

- DAX (Data Analysis Expressions) – Calculated metrics and measures

## Recommendations

- Monitor salary fairness across genders to ensure equal pay.
- Implement employee development and training programs to improve performance.
- Review departments with salary gaps and take corrective if needed.

## Conclusion 

The HR Analytics Dashboard gives a clear overview of employee data at Palmoria Group. By analyzing employee demographics, salaries, and performance, HR and management can make informed decisions to improve staff management and overall organizational performance.
